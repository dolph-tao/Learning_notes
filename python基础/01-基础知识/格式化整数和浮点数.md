# 格式化整数和浮点数

## 格式化整数
```py
n = 1234
print(format(n, '10d'))   # 保留10位输出，自动补空格
print(format(n, '0>10d')) # 左侧补零
print(format(n, '0<10d')) # 右侧补零
```

## 格式化浮点数
```py
x1 = 1234.56789
x2 = 30.1
print(format(x1, '0.2f'))  # 保留小数点后两位
print(format(x2,'0.2f'))
```

## 描述format函数
```py
print(format(x2,'*>15.4f'))   # >右对齐，默认空格，指定则为星号********30.1000
print(format(x2, '*<15.4f'))  # <左对齐，30.1000********
print(format(x2, '*^15.4f'))  # 中心对齐，****30.1000****
print(format(123456789,','))  # 用千位号分隔，123,456,789
print(format(1234567.865565656,',.2f'))  # 前面用千位号分割，小数点保留2位  1,234,567.87

print(format(x1,'E')) # 科学计数法
print(format(x1,'0.2E')) # 科学计数法保留小数点2位
```
