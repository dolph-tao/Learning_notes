# 列表和字典
```py
# 可以作为字典的key的类型
d = {}
d['name'] = 'Bill'
d[10] = 20
d[True] = False
d[12.3] = 20.1

d[(1,2,3)] = [4,5,6]

class Person:
    pass

p1 = Person()
p2 = Person()

d[p1] = "p1"
d[p2] = 'p2'

print(d)
print(d[12.3])

#d[[1,2,3]] = 3 #错误
#d[{'a:',3}] = 4 # 错误
```
## 因为key是不能变的。但列表和字典的值是可以变化的，一旦变化，就再也找不到对应的value了
